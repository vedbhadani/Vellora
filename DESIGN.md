# Design System: Velora Tech (Cream Aesthetic)
**Project ID:** 7445309269079439304

## 1. Visual Theme & Atmosphere
Light, airy, and warm minimal-luxury. A modern, growth-focused agency design with Apple-level simplicity but utilizing a soft, premium "cream" aesthetic. The site should feel high-end, elegant, inviting, and conversion-focused. It uses a clean, structured grid layout, soft earthy gradients, and very subtle, diffused shadows to create an Awwwards-inspired award-winning aesthetic.

## 2. Color Palette & Roles
*   **Cream Base (Background)**: (#F9F8F6) - Used as the primary light background to convey warmth, softness, and luxury.
*   **Soft Sand (Secondary Background)**: (#EFECE5) - Used for card backgrounds or alternating sections.
*   **Deep Espresso (Primary Text)**: (#2A2724) - Used for primary typography, providing high contrast but softer than pure black.
*   **Muted Clay/Terracotta (Accent)**: (#C08465) - Used sparingly for accents, highlights, primary buttons, and critical interactions to inject a warm, organic premium feel.
*   **Soft Sage (Secondary Accent)**: (#A3B19B) - Used for subtle highlights or success states.

## 3. Typography Rules
High-end typography hierarchy using modern geometric sans-serif or elegant serif fonts (e.g., Playfair Display for headings, Inter for body).
*   **Headings**: Bold or Medium, tight letter-spacing, elegant scale difference between H1, H2, and H3.
*   **Body**: Clean, breathable line-height (e.g., 1.6), and regular weight for maximum readability.
*   **Accents**: Occasional use of uppercase text with wide letter-spacing for sub-labels or categories.

## 4. Component Stylings
*   **Buttons**: Subtly rounded corners or pill-shaped, using Muted Clay for primary CTAs with elegant, smooth hover animations.
*   **Cards/Containers**: Clean grid structure with delicate, whisper-soft diffused shadows. Use Soft Sand or stark White against the Cream background for depth.
*   **Inputs/Forms**: Minimalist. Clean stroke styles with subtle background focus states.

## 5. Layout Principles
*   **Whitespace**: Elegant, generous margins and padding. Let the content breathe.
*   **Alignment**: Strict grid alignment, mobile-first responsiveness.
*   **Structure**: Clearer, more defined boundaries between sections compared to the previous iteration.

## 6. Design System Notes for Stitch Generation
Please ensure all generated UI adheres strictly to this LIGHT premium minimal-luxury aesthetic: Cream base background (#F9F8F6), Deep Espresso typography (#2A2724), and warm Muted Clay (#C08465) accents. Keep the layout highly structured, grid-based, spacious, and use subtle, soft shadows for depth. Ensure high-end typography with clear structural hierarchy.
